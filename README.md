# Sistema Integrado

Para rodar este projeto:

1.  Abra o terminal na pasta `sistema`:
    ```bash
    cd sistema
    ```

2.  Instale as dependências (se necessário):
    ```bash
    npm install
    ```

3.  Rode o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```

4.  Acesse `http://localhost:3000` no navegador.

## Importante

O banco de dados atual é **SQLite** (local). Para deploy na Vercel e acesso compartilhado, será necessário configurar um banco PostgreSQL (Supabase ou Vercel Postgres).
